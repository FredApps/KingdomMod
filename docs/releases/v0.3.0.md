# KingdomMod v0.3.0 - Builder speed option

## What's changed

- Adds a new AnyTrees F1 option: `Builder speed: Slow | Fast`.
- Defaults to `Slow` so fresh installs keep vanilla construction speed.
- `Fast` finishes construction when a builder actually starts working, using the game's own construction completion path.

## Why

This gives players a faster building option without completing paid construction immediately. Builders still need to reach the job first, so payment, job assignment, and builder travel remain vanilla.

## Validation

- Build Release locally.
- Deploy locally with `tools\install-mods.ps1 -NoBuild -NoDefenderExclusion`.
- In game, verify the F1 Mods order: `Builder cowardice`, `Builder speed`, `Guerilla Warfare`.
- Confirm Slow mode remains vanilla and Fast mode completes construction on first builder work.
